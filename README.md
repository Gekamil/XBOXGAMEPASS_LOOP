# XBOXGAMEPASS_LOOP
A bot designed to update the subscription to the Gamepass monthly. Designed by IA


Bienvenido a Xbox Auto Loop.

INSTRUCCIONES DE INSTALACIÓN:

Haz doble clic en "Instalar.bat" y dale permisos de Administrador si te lo pide. Espera a que termine de descargar todo (la ventana verde se cerrará sola).

Abre una consola de comandos (CMD) en esta carpeta y escribe: npm run save-session

Se abrirá un navegador. Inicia sesión con tu cuenta de Microsoft/Xbox. Cuando termines, cierra el navegador.

Haz doble clic en "Abrir_Menu.bat" para iniciar el programa.

Ve a la Opción 4 para configurar tu propio bot de Telegram.
